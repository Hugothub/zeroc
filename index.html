Projet Zeroc

Suivez les étapes suvantes pour déterminer les meilleurs matériaux pour votre chantier

  1. Téléchargez le fichier Dynamo:

    2. Ce fichier va vous permettre de récupérer les données du fichier Revit sur votre ordinateur.

    Télécharger le fichier Dynamo
  2. Ouvrez votre fichier Revit
  3. Lancez et executez Dynamo

    4. Sélectionnez le chemin vers le dossier contenant le fichier dynamo téléchargé dans le lecteur Dynamo de REVIT et exécutez le programme Dynamo.

  4. Téléchargez et appliquez la macro suivante

    5. Vous devez maintenant appliquer une macro qui vise à sélectionner les informations
    qui nous seront utiles dans le document Excel. Si vous rencontrez des difficultés, la démarche est expliquée ici

  5. Téléchargez l'Excel en sortie

    6. Une fois le traitement terminé, téléchargez le fichier Excel résultant ici:


  6. Lancez le traitement des données

    7. Utilisez l'Excel pour analyser les données et déterminer les meilleurs matériaux pour votre chantier.
